# Windows Utility Scripts #

``debug_build.bat`` builds debug versions of the following in ``media-file-date-fixer\target\x86_64-pc-windows-msvc\debug``:
- mfdf_ffl.dll: a Win64 DLL version of the library
- mfdf.exe: a Win64 CLI test runner for the library


``release_build.bat`` builds the release versions of the above in ``media-file-date-fixer\target\x86_64-pc-windows-msvc\release``
